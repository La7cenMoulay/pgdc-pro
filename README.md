# PGDC Official Website

Welcome to the official repository for the **Programming & Graphic Design Club (PGDC)** website. This platform acts as the central hub for our community, showcasing our achievements, events, and dynamic member roster.

## üöÄ Features

### Core Experience
- **Modern UI/UX**: Built with a "Glassmorphism" design aesthetic using React and plain CSS variables for high performance.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Interactive Chatbot**: Integrated FAQ assistant for instant visitor support.

### Gallery & Media
- **Dynamic Gallery**: Categorized events and achievements.
- **Lightbox Viewer**: Immersive full-screen image viewer for event albums.
- **Performance**: Automatic image compression for uploaded assets.

### Administration
- **Secure Dashboard**: Protected management area for board members.
- **Content Management**: Add/Remove members and gallery items directly vs the UI.
- **Drag & Drop**: intuitive upload interface for all media assets.

## üì¶ Installation

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/pgdc-website.git
   cd pgdc-website
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```

## üîê Admin Access

The admin dashboard is located at local route `/admin` (or via hidden triggers).
